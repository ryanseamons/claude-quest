# Parent Guide: Code Quest

## Overview

This course is designed for your 12-year-old to learn coding fundamentals using Claude Code, an AI-powered coding assistant. The course is self-paced and designed to be completed independently, though you're welcome to participate!

---

## Setup Requirements

Before your child begins, please ensure:

### 1. Terminal Access
- On Mac: Terminal app (pre-installed)
- On Windows: PowerShell or install Windows Terminal
- On Linux: Terminal (pre-installed)

### 2. Claude Code Installation
Claude Code needs to be installed. If not already set up:
```bash
npm install -g @anthropic-ai/claude-code
```
(Requires Node.js to be installed first)

### 3. Python Installation
Many projects use Python. Check if it's installed:
```bash
python --version
```
If not installed, download from python.org

## How to Support

### Do:
- Let them struggle a bit before helping (problem-solving is learning!)
- Celebrate their creations, even if simple
- Ask them to show you what they built
- Be available for installation/permission issues

### Don't:
- Take over the keyboard
- Fix their code for them (let Claude help)
- Rush them through levels
- Expect perfection

---

## Safety Notes

- Claude Code runs commands on your computer. Monitor initial sessions
- Teach them not to share personal info in prompts
- Review any code before it accesses internet/files
- The course projects are safe terminal and Python programs

---

## Common Issues & Solutions

**"Command not found: claude"**
- Claude Code may not be installed
- Run: `npm install -g @anthropic-ai/claude-code`

**"Command not found: python"**
- Python needs to be installed
- Download from python.org or use `python3` instead

**"Permission denied"**
- May need admin rights
- Try adding `sudo` before command (with caution)

**Terminal looks confusing**
- Type `clear` to clean it up
- Type `cd ~` to go back home

## What They'll Learn

By the end of this course, your child will:

1. **Technical Skills:**
   - Navigate using command line
   - Run programs from terminal
   - Work with AI coding assistants
   - Create basic Python programs

2. **Problem-Solving Skills:**
   - Breaking problems into steps
   - Debugging (fixing errors)
   - Iterating and improving

3. **Communication Skills:**
   - Writing clear prompts
   - Asking for help effectively
   - Explaining technical concepts
