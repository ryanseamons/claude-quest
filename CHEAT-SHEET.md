# Code Quest Cheat Sheet

Print this out or keep it open while you code!

---

## Terminal Commands

| Command | What It Does | Example |
|---------|--------------|---------|
| `pwd` | Where am I? | `pwd` |
| `ls` | What's here? | `ls` |
| `cd [folder]` | Go to folder | `cd Desktop` |
| `cd ..` | Go back one | `cd ..` |
| `mkdir [name]` | Make folder | `mkdir myproject` |
| `clear` | Clean screen | `clear` |
| `python [file]` | Run Python | `python mygame.py` |

---

## Starting Claude Code

```bash
cd ~/Desktop/code-quest    # Go to your folder
claude                      # Start Claude Code
```

---

## Claude Code Commands

| Command | What It Does |
|---------|--------------|
| `/help` | Show all commands |
| `/clear` | New conversation |
| `Ctrl+C` | Exit Claude Code |

---

## Good Prompts for Claude

### Creating Something New:
```
Create a Python program called "[name].py" that [what it does]
```

### Getting Help:
```
I got this error: [paste error]. What does it mean and how do I fix it?
```

### Learning:
```
Explain this code line by line like I'm 12
```

### Improving:
```
Can you add [feature] to my program?
```

---

## Python Quick Tips

**Run a Python file:**
```bash
python filename.py
```

**If `python` doesn't work, try:**
```bash
python3 filename.py
```

---

## When Things Go Wrong

| Problem | Try This |
|---------|----------|
| Command not found | Check spelling |
| No such file | Use `ls` to see files |
| Permission denied | Ask a parent |
| Python error | Tell Claude the error |
| Stuck/confused | Ask Claude to explain |

---

## Magic Words

- **"Step by step"** - Claude breaks it down
- **"Explain like I'm 12"** - Simpler explanation
- **"Can you show me"** - Get examples
- **"What does this mean"** - Get definitions
- **"Start simple"** - Begin with basics

---

## Remember!

- Mistakes are learning
- Ask lots of questions
- Test your code often
- Have fun building!
