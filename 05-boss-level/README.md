## Building It

### Step 1: Describe It to Claude

Start Claude Code and explain your idea. Be specific! Example:

```
I want to create a [type of program] called "[name]" that [what it does].
The user should be able to [main action].
It should [key feature].
Let's start with just [simplest version] and add more later.
```

### Step 2: Get the Basic Version Working

Don't try to build everything at once! Ask Claude to create the simplest possible version first.

### Step 3: Test and Play

Run your program. Use it. Break it. Find what's weird or missing.

### Step 4: Improve It

Go back to Claude and say:
- "Can you add [feature]?"
- "I want it to also [do this thing]"
- "This part is confusing, can we make it clearer?"

### Step 5: Polish It

Make it feel finished:
- Add a welcome message
- Add instructions for the user
- Make the text fun to read
- Fix any rough edges
